# Toy Robot Simulator

## Prerequites
 - Node (8.9.x) & NPM (5.6.x)

## Installation
 1. npm install
 2. npm install -g jest
 3. npm start (to start)
 4. npm test (to test)
 5. npm dist (create prod version & run)

## NPM Commands

 - npm start
 - npm dist
 - npm test
